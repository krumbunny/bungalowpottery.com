About this website

    Web site design and production by Ken Chin-Purcell.

    This is a static web site hosted on Site44

    The site is tested and built using the Ruby based Middleman system.

    Some client side adjustments are made using jQuery 2.

    The CSS is generated using Sass, the library Bourbon and the grid system Neat.

    The fonts Anisette STD Petite, Museo and Myriad Pro are served by Typekit.

    The HTML pages are written in Haml

    The gallery image tool is available from Nivo Slider.
